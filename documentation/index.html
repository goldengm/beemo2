Getting Started AdminPro - Codeigniter + AdminLTE + Modern Design

AdminPro is easy to use admin panel to create your amazing application with all basic tools that you need. Script is built with Codeigniter and uses AdminLTE theme in modern way.

Main Features

  • Modern Design
  • Easy to use & Customize
  • Users Management System
  • Secured Login & Reset Password System
  • Supports AdminLTE & all its components
  • Custom Helpers & other functions
  • Crud Model
  • Much More...

Installation

Steps

  • Copy the main folder
  • Find and Import the sql file from sql folder
    • db.sql - creates one user by default
  • Open application/config/database.php file & fill the credentials
  • You can now login with username: admin & password: admin
  • All Done !! Ready to Go

Faq's

Can i customize this script ?

Yes, the script can be customized easily.

Can i change theme ?

Yes, Changing theme is possible. However, It will require some knowledge of codeigniter directory structure.

Does it have Crud Operations available ?

Crud Operations are avilable but only for database table functions. Every Model that extends MY_MODEL class can access to crud functions. Documentation will be available soon.
